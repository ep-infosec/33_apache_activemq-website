00225 /* The memory requirements for deflate are (in bytes):
00226             (1 << (windowBits+2)) +  (1 << (memLevel+9))
00227  that is: 128K for windowBits=15  +  128K for memLevel = 8  (default values)
00228  plus a few kilobytes for small objects. For example, if you want to reduce
00229  the default memory requirements from 256K to 128K, compile with
00230      make CFLAGS="-O -DMAX_WBITS=14 -DMAX_MEM_LEVEL=7"
00231  Of course this will generally degrade compression (there's no free lunch).
00232 
00233    The memory requirements for inflate are (in bytes) 1 << windowBits
00234  that is, 32K for windowBits=15 (default value) plus a few kilobytes
00235  for small objects.
00236 */
